# 📊 E-Commerce Sales Analysis (SQL + Python Project)

## 📌 Project Overview
This project analyzes an E-Commerce sales dataset using PostgreSQL and Python.  
The objective was to clean raw data, perform exploratory data analysis, and solve real-world business questions using SQL and Python.

---

## 🛠 Tools & Technologies Used
- PostgreSQL
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Window Functions
- Data Cleaning Techniques

---

## 🗂 Dataset Information
The dataset contains:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit

---

## 🧹 Data Cleaning Steps Performed
- Checked and validated data types
- Removed NULL values
- Removed duplicate records
- Handled negative and invalid values
- Standardized text fields
- Created new columns (Order Month, Order Year, Profit Margin)

---

## 📊 Business Questions Solved

1. What is the Total Sales and Total Profit?
2. What is the Overall Profit Margin?
3. What is the Monthly Sales Trend?
4. Top 5 Products by Sales
5. Top 5 Products by Profit
6. Which Category Makes Highest Sales?
7. Which Category Makes Highest Profit?
8. Which Region Generates Highest Sales?
9. Which Region Generates Highest Profit?
10. Which Products Are Loss Making?
11. Monthly Profit Margin Trend
12. Rank Products Within Each Category (Window Function)
13. Contribution % of Each Category (Advanced SQL)
